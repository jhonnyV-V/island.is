# SliderFormField

### Usage in a template

You can create a DescriptionFormField using the following function `buildDescriptionField`.

<Source
  language="typescript"
  code={dedent(`
    builSliderField({
      id: 'field.id',
      label: {
        singular: 'day',
        plural: 'days',
      },
      min: 1,
      max: 5,
      step: 1,
      showMinMaxLabels: true,
      showToolTip: true,
      trackStyle: { gridTemplateRows: 8 },
      calculateCellStyle: () => {
        return {
          background: 'blue',
        }
      },
    })`)}
/>

The previous configuration object will result in the following component:

You can also use this field into a custom component by using `<SliderFormField field={...} />` with the configuration object defined above.
